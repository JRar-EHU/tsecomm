Учебное одностраничное приложение интернет магазина, реализованное без фреймворков — на чистом TypeScript с компонентным подходом.
Основной функционал:
•	клиентская маршрутизация (Navigo)
•	загрузка данных из DummyJSON API
•	каталог товаров, сортировка, фильтры
•	корзина с localStorage
•	checkout форма с HTML5 валидацией
•	модульная структура, компонентный рендеринг
•	стилизация через Bulma + компонентный CSS
